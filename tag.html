Tabella Tag HTML
Nome Tag Descrizione Esempio
HTML È il tag di costruzione dell'HTML. Se si crea un file HTML, è necessario incorporare tutto il contenuto all'interno del tag HTML <html></html>
DOCTYPE Viene utilizzato per definire il tipo di documento. Viene dichiarato prima del tag HTML, in quanto è l'istruzione per il sito web che dice che il documento che sta per arrivare è di tipo HTML. <!DOCTYPE html>
HEAD Una volta aperto il tag HTML, il tag head dovrebbe essere la prima cosa che appare. È il tag che non contiene alcun testo, ma racchiude tutti gli altri tag. Title, style, meta ecc. sono tutti tag messi al suo interno. Viene utilizzato per descrivere il contenuto del sito web. <head></head>
TITLE Questo tag si usa quando si vuole dare un titolo al contenuto. Anche il tag title si trova sotto il tag head. Non include nessun altro tag al suo interno. All'interno di questo tag si può scrivere solo del testo. <title>questa è l'intestazione</title>
BODY Tutto ciò che si vede sul sito web è scritto all'interno di questo tag. Quindi, il contenuto che si vuole far vedere agli utenti deve essere incorporato nel tag body. Una cosa da ricordare è che questo viene dopo il tag head. <body>questo è il corpo</body>
DIV Il tag div, noto anche come tag division, viene utilizzato per definire la divisione del contenuto. Spesso funge da contenitore per gli altri elementi o tag del file HTML. Di solito viene utilizzato per lo styling del contenuto. <div></div>
SPAN Si usa per avvolgere un testo del file. Si usa fondamentalmente per raggruppare il contenuto in linea del file. <span></span>
A È il tag di collegamento ipertestuale. In questo tag è possibile inserire un link a una pagina o ad un sito web in cui si desidera indirizzare il visitatore. Fa parte anche l'attributo "href", che è molto importante in quanto la destinazione viene passata in questo attributo. <a href="https://www.google.com">google</a>
TABLE È il tag tabella, anche se questo tag non fa alcuna differenza a meno che non vi si inserisca del contenuto. <table></table>
THEAD È un altro dei tag table, usato per raggruppare il contenuto dell'intestazione della tabella in un file. Rappresenta la riga superiore della tabella che fornisce l'intestazione delle colonne. <thead></thead>
TBODY È un altro dei tag di tabella utilizzato per raggruppare il contenuto del corpo della tabella in un file. Si usa all'interno dei tag di tabella e viene utilizzato per mostrare i dati effettivi della tabella (si usa dopo il tag "<thead>" (se presente). <tbody></tbody>
TR Questo tag viene passato all'interno del tag table. Viene utilizzato per dichiarare la riga della tabella. Anche questo tag è nullo finché non vi vengono passati i tag che passano il contenuto. Contiene almeno un elemento <td> o <th>. <tr></tr>
TD Questo tag viene passato all'interno del tag <tr>. Può contenere qualsiasi contenuto al suo interno. Viene utilizzato per definire il contenuto che deve essere passato nella riga della tabella. Il contenuto all'interno è una cella. <td></td>
BR Interruzione di riga che viene utilizzato per interrompere le righe all'interno del file. Tutto ciò che fa è aggiungere spaziatura nel paragrafo e assicurare che la riga successiva appaia nella nuova riga. Non esiste un tag di chiusura per <br> <br>
SCRIPT Quando si vuole definire lo script lato client all'interno del file, si usa il tag script. Di solito contiene l'istruzione di scripting al suo interno o può essere indirizzato verso qualsiasi altro utilizzando l'attributo src. <script></script>
IMG Se vuoi inserire un'immagine nei file HTML, è possibile farlo utilizzando il tag img. È possibile definire l'altezza e la larghezza dell'immagine utilizzando altri attributi al suo interno. Non ha un tag di chiusura. <img src="https://bitarray.io/images.logo.png" alt="logo bitarray">
UL Il tag ul è usato per definire un elenco non ordinato puntato. Si tratta di un elemento a livello di blocco in cui vengono inseriti gli elementi dell'elenco. <ul><li>Elemento</li></ul>
LI Il tag li viene utilizzato insieme al tag ul per passare la gerarchia degli elenchi nel file. Quando si vuole passare un elenco all'interno di un elenco, si utilizza il tag li. <li></li>
HR Questo tag viene utilizzato per rappresentare il cambiamento di argomenti all'interno del contenuto. Inserisce una linea orizzontale. <hr>
STRONG Il tag strong è usato per esaltare l'importanza del testo. Quello che fa è mettere il testo in grassetto. <strong>questo è testo forte</strong>
